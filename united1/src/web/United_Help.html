Basic Input

The united calculator supports all basic functions of a calculator! You can easily input numbers, and do simple arithmetic.

Below is a list of commonly used buttons as shown in the picture:

History
View history of past calculations
Input Bar
View numbers and operators inputed as well as type them in manually
Menu Bar
Access advanced features of the calculator
Number Pad
Input numbers
Operators
Evaluate expressions with operands
Unit Dropdown
Select units as well as input your own units

Advanced Input

The united calculator also supports several more advanced functions!

Below is a list of some more advanced buttons as shown in the picture:

About
Open an about page
Add
Add a custom unit and define like units
Colors
Change the colors of the calculator
Convert
Convert displayed units to another like unit
Cut, Copy, & Paste
Basic text editing commands
Logo
Select the image for the logo
Playback
Playback a sequence of calculations
Print
Open a printable version of the history
Record
Record a sequence of calculations in a file
Web
Open an online help page
